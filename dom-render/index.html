Heavy DOM operation

En este ejemplo se puede observar cómo al renderizar una gran cantidad de elementos en el DOM bloqueamos el hilo principal del navegador.

Para ello se leerá una csv con miles de datos que renderizaremos en una tabla.

Una posible solución sería renderizar 1..n elementos a la vez, en vez del total de ellos. Para solucionarlo se hará uso de la función nativa requestAnimationFrame.

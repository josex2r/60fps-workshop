60fps Workshop

Requisitos

  • git
  • nodeJS
  • harp (npm install -g harp)

¿Cómo ejecutarlo?

Asegúrate que primero cumples los requisitos. A continuación, clona y accede al siguiente repositorio:

$ git clone https://github.com/josex2r/60fps-workshop.git && cd 60fps-workshop

Muévete al directorio en donde has clonado el proyecto e instala los paquete con npm o yarn:

$ npm install

Arranca el proyecto ejecutando lo siguiente:

$ npm run server

Y por último accede en tu navegador Chrome alocalhost:9000

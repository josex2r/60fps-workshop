60fps Workshop

Requisitos

  • git
  • nodeJS
  • harp (npm install -g harp)

¿Cómo ejecutarlo?

Asegúrate que primero cumples los requisitos, después clona el siguiente repositorio:

$ git clone https://github.com/josex2r/60fps-workshop.git

Muévete al directorio en donde has clonado el proyecto e instala los paquete con

npmoyarn:$ npm install

Ejecuta:

$ npm server

Y por último accede a localhost:9000

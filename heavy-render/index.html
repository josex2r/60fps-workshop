Heavy Render

En este caso veremos como optimizar operaciones pesadas de rendering para que vayan más fluidas.

El problema consiste en un conjunto de datos que se tienen que el usuario tiene que ver actualizar en tiempo real.

Para optimizarlo haremos que únicamente se rendericen los elementos que el usuario está viendo en ese momento.
